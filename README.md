# Python
Number Guessing Game

Use of loops, conditionals, and user input.

Key Concepts
import random & randint(): Generates a random target number.
input() & int(): Takes user input and converts it to an integer.
while Loop: Runs the game until attempts run out.
if/else: Checks if the guess is high, low, or correct.
try-except: Handles non-integer inputs without crashing.
